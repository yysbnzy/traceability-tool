# 双向溯源分析工具 v2.1

## 更新说明

### v2.1 更新内容（2026-04-24）

1. **新增合并单元格显示模式**
   - A/B 列：用例→需求（合并单元格竖向展开）
   - D/E 列：需求→用例（合并单元格竖向展开）
   - G/I 列：用例→需求（逗号分隔参考版）
   - K/L 列：需求→用例（逗号分隔参考版）

2. **输出格式优化**
   - 合并单元格区域：每个用例/需求独立显示，合并主键列
   - 参考区域：保留原版的逗号分隔格式便于快速查阅
   - 孤儿用例/孤儿需求自动归类至"异常项"Sheet

## 功能说明

### 输入格式
- Excel文件，两列：用例ID | 需求ID
- 支持多行重复用例ID（自动聚合）
- 支持多种分隔符：逗号、分号、中文逗号、换行

### 输出格式
**溯源分析 Sheet**
| 区域 | 列 | 格式 | 说明 |
|------|-----|------|------|
| 用例→需求 | A/B | 合并单元格 | 竖向展开，每个需求一行 |
| 需求→用例 | D/E | 合并单元格 | 竖向展开，每个用例一行 |
| 用例→需求 | G/I | 逗号分隔 | 参考版，单行显示 |
| 需求→用例 | K/L | 逗号分隔 | 参考版，单行显示 |

**异常项 Sheet**
- 孤儿用例：有用例ID但无关联需求
- 孤儿需求：有需求ID但无关联用例
- ID前缀异常：不符合指定前缀规则

### 使用步骤
1. 准备Excel文件（用例ID + 需求ID）
2. 运行程序，选择输入文件
3. 可选：设置用例ID/需求ID前缀过滤
4. 点击"开始分析"
5. 输出文件自动生成（同目录，文件名加 `_output.xlsx`）
