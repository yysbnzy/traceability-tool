# 用例需求溯源分析工具 v3.1

## 功能简介

双向溯源分析工具，支持用例文档与需求文档的关联分析，生成合并单元格和逗号分隔两种显示格式的溯源表。

## 更新记录

### v3.1 (2026-04-24)

**新增功能**
- **合并单元格显示**：用例→需求、需求→用例支持合并单元格竖向展开
- **双Sheet输出**：
  - Sheet1: 双向溯源表-合并版（A/B列合并单元格 + D/E列合并单元格）
  - Sheet2: 双向溯源表-逗号版（A/B列逗号分隔 + D/E列逗号分隔）

## 使用步骤

1. **准备用例文档**
   - Excel 格式，至少两列：用例ID、需求ID
   - 多个需求可用逗号、分号、换行分隔
   - 同一用例可分多行填写，自动聚合

2. **运行程序**
   - 双击 `TraceabilityTool_v3.1_fix3.exe`
   - 点击"浏览"选择用例文档
   - 选择 Sheet 页（通常自动识别）
   - 设置用例ID列和需求ID列（默认 A、B）

3. **配置过滤（可选）**
   - 用例ID过滤：只显示此前缀开头的用例（如 `v7.10#`）
   - 需求ID过滤：只显示此前缀开头的需求（如 `SyRD-VA`）
   - 留空表示不过滤

4. **开始分析**
   - 点击"🚀 开始溯源分析"
   - 选择保存位置，生成报告

5. **查看结果**
   - 双向溯源表-合并版：合并单元格显示
   - 双向溯源表-逗号版：逗号分隔显示
   - 输入源：原始数据完整副本
   - 异常分析：孤儿用例、孤儿需求、被过滤项
   - 需求缺失用例：需求文档有但用例未覆盖（需导入需求文档）
   - 统计汇总：各类数据统计

| 区域 | 列 | 格式 |

| 用例→需求 | A/B | 合并单元格，每个需求一行 |

## 统计项说明

| 统计项 | 说明 |
|--------|------|
| 输入源去重用例数 | 用例文档中不重复的用例ID总数 |
| 输入源去重需求数 | 用例文档中不重复的需求ID总数（含孤儿需求） |
| 输入源去重总数 | 用例ID + 需求ID 总和 |
| 双向溯源表去重用例数 | 过滤后双向溯源表中的用例数量 |
| 双向溯源表去重需求数 | 过滤后双向溯源表中的需求数量 |
| 异常分析表条目数 | 异常分析表中所有异常条目总数 |
| 输出数据总计 | 双向溯源表 + 异常分析表 |
| 总用例数 | 用例文档中用例ID非空的总数 |
| 双向关联用例数 | 有关联需求的用例数 |
| 孤儿用例数 | 未关联任何需求的用例数 |
| 需求被过滤用例数 | 关联需求不符合过滤条件的用例数 |
| 用例中引用需求数 | 用例文档中引用的需求ID总数 |
| 双向关联需求数 | 有关联用例的需求数 |
| 孤儿需求数 | 有用例ID为空但引用的需求数 |
| 需求文档总需求数 | 需求文档中的需求总数（需导入需求文档） |
| 需求缺失用例数 | 需求文档中有但用例未覆盖的需求数 |
| 需求覆盖率 | 已覆盖需求数 / 需求文档总需求数 |

## 常见问题

**Q: 过滤后双向溯源表为空？**
A: 检查过滤条件是否包含隐藏字符（如换行符、零宽空格），建议手动输入而非粘贴。

**Q: 需求缺失用例为空？**
A: 需要导入独立的需求文档才能计算覆盖率。未导入需求文档时此 Sheet 为空。

**Q: 孤儿需求是什么？**
A: 用例文档中 B 列写了需求ID，但 A 列（用例ID）为空的行。
