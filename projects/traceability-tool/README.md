# 用例-需求溯源分析工具

根据 Excel 表格中的**用例ID**和**需求ID**列，自动生成双向溯源矩阵。

## 功能

- **A/B 列**：保留原始输入数据
- **D/E 列**：用例ID → 聚合后的所有关联需求ID（一行显示）
- **G/H 列**：需求ID → 聚合后的所有关联用例ID（一行显示）
- **智能分隔符解析**：支持中文逗号 `，`、英文逗号 `,`、分号 `;`、句号 `。`、空格、换行等多种分隔形式
- **自动去重排序**：每个用例/需求只出现一次，关联项去重并按字母排序
- **孤儿用例检测**：提示没有关联任何需求的用例

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方式

### 方式一：GUI 界面（推荐）

双击运行或命令行启动：

```bash
python gui.py
```

界面操作：
1. 点击 **"选择文件"** 按钮，选择输入的 Excel 文件
2. 点击 **"开始分析"** 按钮
3. 分析完成后，输出文件会自动保存在原文件同目录下，文件名后缀为 `_output.xlsx`

### 方式二：命令行

```bash
python traceability.py <输入Excel路径> <输出Excel路径>
```

示例：
```bash
python traceability.py input.xlsx output.xlsx
```

## 输入格式要求

Excel 文件**第一列**为 `用例ID`，**第二列**为 `需求ID`。

支持两种输入形式：

1. 一个用例ID一行，需求ID列中用逗号分隔多个需求：

| 用例ID | 需求ID |
|--------|--------|
| TC-001 | RD1, RD2 |
| TC-002 | RD2，RD3 |

2. 一个用例ID分多行，每行一个需求ID（也会被自动聚合）：

| 用例ID | 需求ID |
|--------|--------|
| TC-001 | RD1 |
| TC-001 | RD2 |

## 输出格式

生成的新 Excel 包含在一个 Sheet 中：

- **A/B 列**：原始数据
- **D/E 列**：用例 → 需求（去重聚合，逗号分隔）
- **G/H 列**：需求 → 用例（去重聚合，逗号分隔）
