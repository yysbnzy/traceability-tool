# 测试用例与需求溯源工具 v2.0

## 功能概述
双向追溯分析工具，支持用例文档和需求文档分别导入，自动识别列，前缀过滤，ID拼接，生成多维度溯源报告。

## 核心功能
1. **双文档导入** - 分别选择用例文档和需求文档
2. **智能列识别** - 自动读取Excel列标题，下拉框选择
3. **前缀过滤** - 支持用例/需求前缀识别过滤
4. **ID拼接** - 支持前缀/后缀自定义拼接
5. **四维度输出** - Sheet1双向溯源、Sheet2输入源、Sheet3异常分析、Sheet4需求缺失

## 使用说明
```bash
python traceability_v2.py
```

## 界面配置项
1. 用例文档 - 文件选择
2. 用例ID列 - 下拉选择(A-Z)
3. 需求ID列(用例文档内) - 下拉选择(A-Z)
4. 用例前缀识别 - 文本框
5. 需求前缀识别 - 文本框
6. 用例前缀(拼接) - 文本框
7. 用例后缀(拼接) - 文本框
8. 需求前缀(拼接) - 文本框
9. 需求后缀(拼接) - 文本框
10. 需求文档 - 文件选择
11. 需求ID列 - 下拉选择(A-Z)

## 输出格式
### Sheet1 - 双向溯源表
- A列: 用例ID(拼接后)
- B列: 关联需求ID列表
- D列: 需求ID(拼接后)
- E列: 关联用例ID列表

### Sheet2 - 输入源
- 原始用例ID列 + 需求ID列

### Sheet3 - 异常分析
- 孤儿用例、孤儿需求、前缀过滤异常

### Sheet4 - 需求缺失
- 需求文档中存在但无用例关联的需求ID
